JavaScript vs HTML and CSS

HTML is a structural language used to build websites. CSS is a styling language that designs what a website will look like. JavaScript is a more advanced language to make web pages dynamic. These three languages have differences, but they are used in unison to present websites.

Control flow and loops

Control flow relates to the order in which tasks or code is run. For example:

Wake up
Tired => Press snooze
Not tired => Get out of bed
Loops iterate over the same piece of code until the condition turns false. For example:
Go to bathroom and brush teeth
Are teeth clean?
No => Action: Up then down
Yes => Stop Brushing

Difference between accessing data from arrays and object literals

Arrays and object literals can contain items. The difference between them is how those items are stored. With arrays they will just be a list of items: 27, Pakenham St, Cat, however, with object literals we can go a step further to give ID's to each of the object items: Age: 27, Area of Work: Elgin St, Cat or Dog: Cat. They also differ in how we access those values. With Object literals, you access their values using dot notation (console.log(person.location);), with arrays, you access the values using numerical index (console.log(array[2]);)

What are functions and why are they useful

Functions are a block of code that can be called upon to perform a task or calculate a value. It can take arguments, manipulate arguments/set variables and/or return a variable. It helps make code easier to read and saves you from repeating yourself.
